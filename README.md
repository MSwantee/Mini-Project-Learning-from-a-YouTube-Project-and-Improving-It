# Mini-Project-Learning-from-a-YouTube-Project-and-Improving-It

## Overview
This project is a dynamic Quiz Application built using HTML, CSS, and JavaScript. It was created as part of a Mini Project to demonstrate the ability to learn from existing tutorials, understand code structure, and implement custom improvements.

## Reference Material
* **Tutorial Video:** https://www.youtube.com/watch?v=vZe4SvsnLxM&list=PLpwngcHZlPadhRwryAXw3mJWX5KH3T5L3&index=5
* **Original Creator:** CodingNepal

## Key Features
*   **Dynamic Questions:** Questions are pulled from a separate JavaScript data file for cleaner code.
*   **Interactive UI:** Modern purple gradient design with hover effects and responsive buttons.
*   **Timer System:** Users have 15 seconds to answer each question.

## My Added Improvement 
Beyond the original tutorial, I implemented a **Persistent High Score System** using `localStorage`.
